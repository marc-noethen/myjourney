## Kursstruktur

### Part 1 — ML Foundations
| Modul | Thema | Tools |
|-------|-------|-------|
| 01 | Intro: ML vs. regelbasierte Systeme · CRISP-DM · Setup | Python, NumPy |
| 02 | Regression: Auto-Preis Vorhersage · Feature Engineering · Regularization | NumPy, Pandas, Scikit-Learn |
| 03 | Classification: Churn Prediction · Logistic Regression · Feature Importance | Scikit-Learn, Matplotlib |
| 04 | Evaluation: Accuracy · Precision · Recall · ROC-AUC · Cross Validation | Scikit-Learn |
| 05 | Deployment: Modell als Service · FastAPI · uv · Docker · Cloud | FastAPI, Docker |
| 06 | Trees & Ensembles: Decision Trees · Random Forest · XGBoost · Gradient Boosting | XGBoost, Scikit-Learn |

### Midterm Project
