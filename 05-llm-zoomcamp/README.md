| Modul | Thema | Tools |
|-------|-------|-------|
| 01 | Introduction to LLMs and RAG | OpenAI API · Elasticsearch |

## Was du baust (Capstone Project)
1. Durchsuchbare Knowledge Base aus eigenem Datensatz
2. RAG Pipeline — Retrieval → Prompt → LLM → Antwort
3. Evaluation — Retrieval-Qualität & Antwortgenauigkeit messen
4. User Interface — Streamlit oder FastAPI
5. Monitoring — Queries, Feedback & Performance tracken
